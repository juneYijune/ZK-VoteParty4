# 党组织管理员首页功能说明

## 功能概述

党组织管理员首页用于展示当前党组织的投票总体情况，帮助管理员快速了解组织内投票活动运行状态。

## 主要功能

### 1. 党组织信息展示
- **党组织名称**：显示当前党组织名称（如果已设置）
- **区块链地址**：显示党组织管理员的钱包地址（缩略显示）

### 2. 统计卡片（3个）
- **已创建投票**：显示该党组织创建的所有投票数量
- **进行中投票**：显示当前正在进行的投票数量
- **已结束投票**：显示已经结束的投票数量

### 3. 最近创建的投票列表
显示最近创建的5条投票记录，包括：
- 投票标题
- 投票类型（差额选举/决议表决/评议评价）
- 投票状态（投票已开启/投票已关闭/投票未开启）
- 投票ID
- 最多可选项数
- 选项数量

## 数据来源

### 链上数据
使用合约方法获取数据：

1. **getVotesByPartyOrg(address)**
   - 获取指定党组织创建的所有投票ID列表
   - 参数：党组织管理员地址
   - 返回：投票ID数组

2. **getVoteInfo(voteId)**
   - 获取指定投票的详细信息
   - 参数：投票ID
   - 返回：投票详细信息（标题、类型、状态、时间等）

### 状态判断逻辑

**投票状态判断：**
```javascript
var now = Math.floor(Date.now() / 1000);

if (status === 2 || now > endTime) {
  // 投票已关闭
} else if (status === 1 && now >= startTime && now <= endTime) {
  // 投票已开启
} else {
  // 投票未开启
}
```

## 使用方式

1. **访问首页**
   - 登录党组织管理员账户
   - 自动跳转到首页 `/partyorg/home`
   - 或点击左侧导航栏"首页"

2. **查看统计信息**
   - 页面顶部显示党组织信息
   - 中间显示3个统计卡片
   - 底部显示最近创建的投票列表

3. **数据刷新**
   - 页面加载时自动获取最新数据
   - 数据来自区块链，确保准确性

## 注意事项

1. **钱包连接**
   - 需要连接 MetaMask 钱包
   - 确保使用党组织管理员地址登录

2. **网络切换**
   - 确保钱包连接到正确的网络
   - 网络不匹配会提示错误

3. **数据加载**
   - 首次加载可能需要几秒钟
   - 投票数量较多时加载时间会更长

4. **党组织名称**
   - 名称从本地存储读取
   - 如果未设置，显示"党组织管理员"
