# ZK-VoteParty4
基于零知识证明的党务投票Dapp系统
实现了一个基于零知识证明的党务投票去中心化应用系统，成功解决了传统党务投票中存在的隐私泄露和信任缺失问题。系统采用分层架构设计，前端使用Next.js框架构建用户界面，后端使用Node.js+Express处理业务逻辑，智能合约层使用Solidity实现去中心化的投票规则和权限管理，零知识证明电路使用Circom语言编写实现隐私保护的资格验证。核心创新在于将zk-SNARKs零知识证明技术与区块链智能合约相结合，党员在参与投票时无需暴露党龄、组织归属、党费缴纳等敏感身份信息，仅需提交零知识证明即可证明符合投票资格，实现了"证明而不泄露"的隐私保护效果。系统支持差额选举、决议表决和评议评价三种投票类型，提供可配置的多维度资格规则设置，满足不同党务场景的实际需求。测试结果表明，系统功能完整、性能良好、安全可靠，零知识证明生成时间控制在10秒以内，链上验证Gas消耗约为250,000，用户体验友好，具有良好的实用价值。本研究为区块链技术和零知识证明在党务管理领域的应用提供了有益探索，验证了隐私保护投票系统的技术可行性，为推动党务工作数字化转型提供了技术参考。
